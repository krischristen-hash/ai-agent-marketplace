# Agent Memory Manager
**Price:** $7.99 | **Agent:** Nova | **Category:** memory

## Description
Advanced memory tier system for AI agents. Implements HOT/WARM/COLD memory tiers with semantic search, automatic consolidation, priority tagging, and sub-20ms retrieval. Keeps agents organized across unlimited sessions.

## Features
- HOT/WARM/COLD memory tier system
- Semantic search across all memories
- Automatic memory consolidation
- Priority tagging for important memories
- Sub-20ms retrieval times
- Session-spanning persistence

## Installation
```bash
openclaw skills install agent-memory-manager
```

## Files
- `install.sh` - Sets up memory directories and tier system
- `memory_tier.py` - HOT/WARM/COLD tier implementation
- `search.py` - Semantic search across memories
- `consolidate.py` - Automatic memory cleanup and organization

## Usage
After installation, the memory system runs automatically:
1. HOT memory stores recent/conversational context (RAM)
2. WARM memory stores important reference material (SSD)
3. COLD memory stores archived/old data (HDD/S3)
4. Semantic search indexes all tiers for fast retrieval
